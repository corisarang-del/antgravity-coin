# Ant Gravity Design System Master

## 온보딩 요약
- 이 문서는 앤트그래비티 웹 경험의 공통 기준이다.
- 진입 구조는 `넷플릭스형 랜딩 → 데이터 중심 홈`이다.
- 구현 전 확인 순서: 공통 규칙 → 토큰 → 랜딩/앱 공통 컴포넌트 → 화면 오버라이드.

## 제품 성격
- 웹뷰 기반 AI 코인 예측 배틀 제품
- 첫 인상은 강렬한 엔터테인먼트, 실제 플레이는 데이터 중심
- 랜딩은 시네마틱하고, 홈부터는 분석 SaaS처럼 빠르게 전환한다

## 톤 앤 매너
- 키워드: cinematic, collectible, playful editorial, trusted analysis
- 넷플릭스처럼 대형 히어로와 가로 섹션 구조를 사용한다
- 공통 컴포넌트와 색감은 `Wall of Love` 계열의 크림/로즈/버터 옐로 감각을 차용한다
- 금지 카피: 수익 보장, 확정 매수·매도 유도, 공포 조장 문구

## 컬러 방향
- 앱 기본 배경은 따뜻한 크림과 잉크 네이비 기반이다
- 랜딩 전용 강조색은 로즈 핑크, 버터 옐로, 하늘빛 블루를 보조로 사용한다
- 금융 상태색은 기존 의미를 유지한다
  - bull: 상승
  - bear: 하락
  - xp: 보상

## 레이아웃 원칙
- 랜딩은 넷플릭스형 구조를 따른다
  - 대형 Hero
  - 가로 섹션형 카드 배열
  - 명확한 CTA 2개 이내
- 홈부터는 모바일 우선 단일 컬럼 + 데이터 카드 중심으로 전환한다
- 감성은 랜딩에서 강하게, 분석은 홈부터 강하게

## 컴포넌트 규칙
- 버튼, 배지, 카드, 패널 등 공통 컴포넌트는 파스텔 포인트를 허용한다
- 하지만 검색, 지표, 결과 카드 등 핵심 데이터 UI는 가독성을 우선한다
- 카드 반경은 크고 부드럽게, 텍스트 대비는 항상 충분해야 한다

## 타이포그래피
- Display: Space Grotesk
- Body: Plus Jakarta Sans / Pretendard
- 랜딩 헤드라인은 압축된 대형 타이포
- 홈과 앱 본문은 16px 이상, 숫자 데이터는 semibold + tabular-nums

## 접근성 / 상호작용
- 모바일 375px 기준 가로 스크롤 금지
- 모든 클릭 요소 최소 44x44px
- 포커스 링 항상 유지
- 애니메이션은 transform/opacity 위주, 과한 장식 금지

## 보안 / 신뢰
- GPS 등 민감한 개인정보 저장 금지
- 결과 화면에는 근거 지표와 리스크 문구를 반드시 같이 둔다
- 캐릭터는 분석 가이드 역할만 수행하고 직접적인 투자 유도는 금지한다

## 품질 체크
- 랜딩과 홈의 톤 전환이 부자연스럽지 않은지 확인
- 공통 토큰이 랜딩/앱 양쪽에서 충돌 없이 동작하는지 확인
- 라이트/다크 모두 대비 기준 충족
