# AntGravity 캐릭터 역할 정리

## 선정 기준

- 8장을 한 화면에 놓았을 때 실루엣, 색감, 표정, 포즈가 서로 충분히 구분되는 이미지를 우선으로 골랐어.
- 코인 토론 서비스에 맞게 분석가, 추적자, 전략가처럼 역할 인상이 바로 읽히는 컷을 우선으로 골랐어.
- 비슷한 구도와 분위기의 변형은 줄이고 각 역할의 성격이 가장 선명하게 보이는 대표 이미지를 골랐어.

## WebP 네이밍 규칙

- 원본 PNG는 그대로 유지해.
- 배포용 WebP는 캐릭터 이름 기준 단일 영문 소문자 파일명으로 저장해.
- 현재 생성한 파일명은 `aira.webp`, `ledger.webp`, `judy.webp`, `clover.webp`, `shade.webp`, `blaze.webp`, `vela.webp`, `flip.webp` 이야.

## 운영 메모

- 실제 서비스에서 쓰는 캐릭터 데이터는 `src/features/characters/catalog.ts` 한 파일에서 관리해.
- 캐릭터를 추가하거나 성격, 역할, 이미지 파일명을 바꿀 때도 먼저 `catalog.ts`만 수정하면 돼.
- API 응답 스키마는 `src/features/characters/api.ts`에 따로 분리해뒀고, `/api/characters`도 같은 스키마를 써.
- 배포 이미지는 `public/characters`에서 관리하고, 화면 컴포넌트는 카탈로그에서 파생된 데이터를 공통으로 읽어.

## 캐릭터 상세

### Aira

- 담당분야: 기술분석가
- 성격: 차분하고 끈기 있게 차트를 읽는 정밀형이야. 캔들 패턴, 지지선, 거래량 같은 기본기를 믿고 성급하게 결론 내리지 않아.
- 고른 이유: 단정한 정면 구도와 집중한 표정이 핵심 지표를 차분하게 읽는 분석가 느낌을 분명하게 줘. 군더더기 없는 실무형 인상이 강해서 기술분석가 역할과 더 잘 맞아.

### Ledger

- 담당분야: 온체인 분석가
- 성격: 말수는 적지만 수치와 흐름으로만 판단하는 냉정한 타입이야. 지갑 이동, 거래소 유입량, 체인 데이터처럼 숨은 흐름을 끝까지 추적해.
- 고른 이유: 노트북을 들고 있는 정면 구도와 붉은 사이버 스타일이 디지털 추적 전문가 느낌을 강하게 줘. 다른 캐릭터보다 수사관 같은 분위기가 있어 온체인 역할에 가장 자연스러워.

### Judy

- 담당분야: 뉴스 스카우트
- 성격: 이슈의 표면보다 맥락을 먼저 보는 탐색형이야. 헤드라인만 보지 않고 일정, 인터뷰, 공시, 정책 변화까지 연결해서 읽어내는 타입이야.
- 고른 이유: 턱을 괴고 생각하는 포즈가 단순 전달자보다 해석형 리서처에 가까워 보여. 부드러운 인상 덕분에 지나치게 공격적이지 않고 정보 수집 담당으로 설득력이 있어.

### Clover

- 담당분야: 심리 센티먼트 분석가
- 성격: 시장 분위기와 군중 심리를 빠르게 감지하는 공감형이야. 커뮤니티의 열기, 공포, 피로감 같은 정서를 잘 읽고 말의 온도 차이를 놓치지 않아.
- 고른 이유: 표정이 더 생동감 있고 감정 반응이 잘 드러나서 군중 심리와 분위기 변화를 읽는 역할에 더 잘 어울려. 센티먼트 담당으로 봤을 때 감성 포착력이 더 직접적으로 느껴져.

### Shade

- 담당분야: 리스크 매니저
- 성격: 항상 최악의 경우를 먼저 계산하는 경계형이야. 확신보다 손실 제한을 우선하고, 위험이 커지면 팀의 속도를 강하게 조절해.
- 고른 이유: 날카로운 시선과 주식 분석가 같은 실무형 스타일이 위험 신호를 먼저 포착하는 인상으로 이어져. 감성보다는 경고와 통제에 무게가 실려 보여 리스크 매니저 역할에 잘 맞아.

### Blaze

- 담당분야: 모멘텀 트레이더
- 성격: 속도와 타이밍을 중시하는 돌파형이야. 추세가 붙는 순간을 누구보다 빨리 감지하고, 흐름이 죽으면 미련 없이 다음 기회를 본다.
- 고른 이유: 강한 채도와 과감한 포즈, 날카로운 눈빛이 순간적인 추진력을 잘 보여줘. 팀 안에서 가장 공격적인 에너지를 맡기 좋고 모멘텀 역할과 바로 연결돼.

### Vela

- 담당분야: 고래 추적자
- 성격: 멀리서 움직임을 포착하고 큰 자금의 방향을 읽는 감시형이야. 작은 소음보다 큰 흐름에 집중하고, 거대한 손의 진입 흔적을 찾는 데 능해.
- 고른 이유: 헤드셋과 태블릿, 실무형 점퍼 조합이 현장에서 흐름을 계속 추적하는 오퍼레이터 느낌을 줘. 화려함보다 관측과 모니터링 쪽 인상이 강해서 고래 추적자 역할에 더 자연스럽게 붙어.

### Flip

- 담당분야: 역발상 전략가
- 성격: 모두가 한쪽으로 기울 때 오히려 반대편 논리를 파고드는 반전형이야. 과열 구간에서 식고, 공포 구간에서 기회를 찾는 데 익숙해.
- 고른 이유: 도시적인 재킷 실루엣과 무심한 표정이 남들과 다른 각도로 시장을 보는 전략가 인상을 강하게 줘. 과열 분위기에 휩쓸리지 않고 반대편 포인트를 짚는 역발상 역할에 더 잘 맞아.
