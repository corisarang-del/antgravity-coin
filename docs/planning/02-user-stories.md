# 유저 스토리

- 작성시각: 2026-03-21 03:01 KST
- 기준: 현재 라우트와 클라이언트 저장 흐름

## 페르소나

### P1. 코린이 초보 사용자

- 지표를 길게 읽기 어렵다.
- 한 화면에서 bull과 bear 근거를 비교하고 싶다.
- 복잡한 리포트보다 캐릭터형 토론이 더 이해하기 쉽다.

### P2. 빠른 판단이 필요한 중급 사용자

- 검색 후 바로 battle에 들어가고 싶다.
- 핵심 논거가 모이면 전체 토론을 끝까지 기다리지 않고 선택하고 싶다.
- 결과를 나중에 다시 와서 복기하고 싶다.

### P3. 로그인 사용자

- 게스트 상태에서 보던 기록을 계정으로 옮기고 싶다.
- 내 battle 기록과 결과를 `/me`에서 다시 보고 싶다.

### P4. 운영자

- 최근 battle outcome과 report source를 빠르게 보고 싶다.
- reusable memo와 character lesson을 battleId 기준으로 다시 보고 싶다.

## 현재 구현 기준 메모

- 랜딩은 `/`, 실제 앱 탐색 시작점은 `/home`
- 시간프레임은 `5m`, `30m`, `1h`, `4h`, `24h`, `7d`
- 토론은 4라운드 병렬 구조
- `battle_pick_ready` 이후 바로 pick 이동 가능

## 유저 스토리

### 탐색

**US-01**
```text
사용자로서,
/home에서 코인 티커나 이름을 검색하면
바로 battle로 들어갈 후보를 찾고 싶다.
```

**US-02**
```text
사용자로서,
Top 코인 목록이 보이면
검색 없이도 바로 battle을 시작하고 싶다.
```

**US-03**
```text
사용자로서,
최근 본 코인이 남아 있으면
이전에 본 battle 맥락으로 쉽게 돌아가고 싶다.
```

### battle 시청

**US-04**
```text
사용자로서,
battle 화면에 진입하면
시장 요약과 함께 캐릭터 발언이 실시간으로 쌓이는 걸 보고 싶다.
```

**US-05**
```text
사용자로서,
발언 전에 미니 브리핑과 곧 등장할 캐릭터 카드가 보이면
기다리는 동안 지금 무슨 준비가 진행되는지 알고 싶다.
```

**US-06**
```text
사용자로서,
bull 2명과 bear 2명의 핵심 논거가 모이면
8명 전체 완료 전에도 선택 단계로 먼저 넘어가고 싶다.
```

**US-07**
```text
사용자로서,
각 발언에 요약과 세부 설명, 대표 지표가 붙어 있으면
왜 그런 주장을 하는지 빠르게 이해하고 싶다.
```

### 선택과 결과

**US-08**
```text
사용자로서,
pick 화면에서 bull / bear와 6개 시간프레임 중 하나를 고르고
실제 정산 시각 기준으로 결과를 기다리고 싶다.
```

**US-09**
```text
사용자로서,
waiting 화면에서 내 선택과 남은 시간을 확인하면
언제 다시 결과를 보러 와야 하는지 알 수 있다.
```

**US-10**
```text
사용자로서,
result 화면에서 승패와 XP 변화, 승리 팀 하이라이트를 보면
내 판단이 왜 맞았거나 틀렸는지 복기하고 싶다.
```

**US-11**
```text
사용자로서,
같은 결과가 한 번만 적용되면
새로고침이나 재방문으로 XP가 중복 반영되지 않아야 한다.
```

### 인증과 아카이브

**US-12**
```text
로그인 사용자로서,
게스트 상태의 level, snapshot, userBattle, 기존 battle 자산이 계정으로 병합되면
기록을 잃지 않고 계속 쓰고 싶다.
```

**US-13**
```text
로그인 사용자로서,
/me에서 최근 battle과 특정 battle 상세를 다시 보면
과거 판단과 캐릭터 lesson을 복기하고 싶다.
```

### 신뢰와 안전

**US-14**
```text
모든 사용자로서,
이 서비스가 투자 조언이 아니라는 고지를 주요 화면에서 보면
콘텐츠 성격을 오해하지 않고 사용하고 싶다.
```

### 인증 UX / 대시보드 고도화 추가

**US-15**
```text
처음 온 사용자로서,
/login에서 Google이나 Kakao 버튼만 눌러 바로 가입하고 로그인되면
복잡한 회원가입 폼 없이 빠르게 시작하고 싶다.
```

**US-16**
```text
로그인 사용자로서,
/me 첫 화면에서 프로필, XP, 등급, 승패 기록을 먼저 보면
내 현재 상태를 한눈에 확인하고 싶다.
```

**US-17**
```text
로그인 사용자로서,
/me에서 최근 battle 기록은 짧게 훑고 필요할 때만 상세를 열면
내 페이지가 길게 늘어지지 않고 핵심 정보부터 보고 싶다.
```
