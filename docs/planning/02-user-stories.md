# 유저 스토리

- 작성시각: 2026-03-21 18:27 KST
- 기준: 현재 라우트와 클라이언트 흐름

## Persona

### P1. 코인 초보 사용자

- 숫자보다 설명과 비교가 먼저 필요하다.
- 캐릭터 토론 형식으로 근거를 빠르게 읽고 싶다.

### P2. 빠른 판단이 필요한 사용자

- 검색 후 바로 battle에 들어가고 싶다.
- 8명 전체를 기다리지 않고 조기 선택하고 싶다.

### P3. 로그인 사용자

- 게스트로 보던 battle 기록을 계정에 붙이고 싶다.
- 과거 battle 결과와 캐릭터 교훈을 다시 보고 싶다.

### P4. 운영자

- 최근 outcome과 report source를 빠르게 보고 싶다.
- battleId 기준 상세, event log, memo를 다시 보고 싶다.

## 탐색

### US-01

사용자로서 `/home`에서 코인을 검색해서 바로 battle에 진입하고 싶다.

### US-02

사용자로서 Top 코인 카드에서 검색 없이 battle을 시작하고 싶다.

### US-03

사용자로서 최근 본 코인 목록으로 다시 돌아가고 싶다.

## battle 시청

### US-04

사용자로서 battle 화면에서 시장 요약과 실시간 토론 메시지를 함께 보고 싶다.

### US-05

사용자로서 첫 메시지 전에도 미니 브리핑과 다음 캐릭터 예고를 보며 기다리고 싶다.

### US-06

사용자로서 bull 2개, bear 2개 근거가 모이면 8명 전부를 기다리지 않고 pick 단계로 가고 싶다.

### US-07

사용자로서 각 메시지의 summary, detail, indicator를 같이 보고 왜 그런 주장을 하는지 이해하고 싶다.

## 선택과 결과

### US-08

사용자로서 pick 화면에서 bull / bear와 시간프레임을 고르고 결과를 기다리고 싶다.

### US-09

사용자로서 waiting 화면에서 남은 시간과 내 선택을 같이 확인하고 싶다.

### US-10

사용자로서 waiting countdown이 끝나면 직접 누르지 않아도 result로 이동하고 싶다.

### US-11

사용자로서 result 화면에서 verdict와 XP를 먼저 보고, report는 늦게 와도 멈추지 않는 흐름을 원한다.

### US-12

사용자로서 같은 결과가 여러 번 적용돼 XP가 중복 반영되지 않기를 원한다.

## 인증과 아카이브

### US-13

로그인 사용자로서 게스트 상태의 local 레벨, recent coins, battle 자산을 계정으로 병합하고 싶다.

### US-14

로그인 사용자로서 `/me`에서 최근 battle 목록과 선택한 battle의 상세를 다시 보고 싶다.

## 운영

### US-15

운영자로서 `/admin/battles`에서 최근 outcome, report source, event log를 보고 싶다.

### US-16

운영자로서 `/admin/memos`에서 reusable memo와 character lesson을 다시 보고 싶다.
