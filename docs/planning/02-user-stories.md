# 유저 스토리

## 페르소나

### P1. 코린이 초보 개미
- 차트와 지표를 자세히 읽기 어렵다.
- 누가 왜 bullish인지 bearish인지 쉽게 비교하고 싶다.
- 딱딱한 리포트보다 캐릭터형 설명을 더 잘 받아들인다.

### P2. 중급 투자자
- 기본 차트는 읽지만 뉴스, 센티먼트, 선물 지표를 한 번에 보고 싶다.
- 빠르게 요약을 보되 필요하면 세부 근거도 같이 확인하고 싶다.

---

## 현재 구현 기준 메모

- 실제 홈 화면은 `/home`이다.
- 랜딩 페이지 `/`는 진입과 브랜드 소개 역할이다.
- 결과 판정은 현재 `24h`, `7d`만 지원한다.
- 종합 점수형 분석보다 실제 팀 선택과 결과 확인 경험이 우선 구현돼 있다.

## 유저 스토리

### 탐색과 진입

**US-01**
```text
사용자로서,
홈에서 코인 티커나 이름을 검색하면
빠르게 배틀로 들어갈 후보를 찾을 수 있다.
```

**US-02**
```text
사용자로서,
Top 코인 목록을 보면
검색 없이도 바로 배틀을 시작할 수 있다.
```

**US-03**
```text
사용자로서,
최근에 본 코인 목록이 남아 있으면
이전에 본 배틀 맥락으로 다시 돌아가기 쉽다.
```

### 배틀 시청

**US-04**
```text
사용자로서,
배틀 화면에 진입하면
시장 데이터 요약과 함께 8명의 발언이 순서대로 쌓이는 걸 보고 싶다.
```

**US-05**
```text
사용자로서,
각 발언에 요약과 세부 설명, 대표 지표가 붙어 있으면
캐릭터가 어떤 근거로 말하는지 이해하기 쉽다.
```

**US-06**
```text
사용자로서,
현재 누가 말하고 있는지 강조되면
실시간 토론처럼 더 몰입해서 볼 수 있다.
```

### 선택과 결과

**US-07**
```text
사용자로서,
토론이 끝나면 bull 또는 bear를 선택하고
24시간 또는 7일 기준으로 결과를 보러 갈 수 있어야 한다.
```

**US-08**
```text
사용자로서,
결과 화면에서 내가 맞았는지와 XP 변화가 바로 보이면
내 판단을 쉽게 복기할 수 있다.
```

**US-09**
```text
사용자로서,
결과가 한 번만 적용되면
새로고침으로 XP가 중복 반영되지 않아야 한다.
```

### 신뢰와 안전

**US-10**
```text
모든 사용자로서,
이 서비스가 투자 조언이 아니라는 고지 문구를
주요 화면에서 항상 볼 수 있어야 한다.
```
