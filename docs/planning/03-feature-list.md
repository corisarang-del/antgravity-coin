# 기능 목록

## 우선순위 기준
- `P0`: 현재 데모 흐름 유지에 필수
- `P1`: 사용감과 설득력 강화
- `P2`: 운영 고도화 또는 후속 확장

## 현재 구현 기준 메모

- 현재 코드는 검색, 배틀, 선택, 결과, 캐릭터 도감까지 연결돼 있다.
- 반대로 실서비스형 장기 대기, 알림, 정교한 종합 점수는 아직 없다.
- 문서는 실제 구현 기준을 우선으로 맞춘다.

## MVP 기준

현재 MVP는 아래를 포함한다.

- 코인 탐색
- SSE 배틀 시청
- 팀/기간 선택
- 결과 계산과 XP 반영
- 결과 중복 적용 방지
- 캐릭터 도감 확인
