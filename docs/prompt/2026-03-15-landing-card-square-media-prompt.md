# 랜딩 캐릭터 카드 정사각 미디어 프레임 적용 프롬프트

- 작성시각: 2026-03-15 16:22 KST
- 해결하고자 한 문제:
  - 랜딩페이지 캐릭터 카드들도 정사각 원본 비율에 맞춰야 했음
- 사용한 프롬프트:
  - 랜딩페이지의 캐릭터 카드들도 그럼 정사각형에 맞게 바꿔줘
- 해결된 것:
  - 랜딩 시즌 라인업 카드 미디어 프레임을 정사각형으로 조정함
  - 카드 폭을 소폭 넓히고 소개영역은 유지함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 라인업 전체 밀도 조정은 추가 가능
