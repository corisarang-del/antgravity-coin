# 캐릭터 토론 설정 파일화 및 ledger.webp 복구 프롬프트

- 작성시각: 2026-03-17 14:25 KST
- 해결하고자 한 문제:
  - 캐릭터별 모델/정보소스/프롬프트/표시 위치를 한 번에 정리하고, 파일 수정만으로 직접 바꿀 수 있게 해야 했다.
  - `ledger.webp`가 없으면 `ledger.png`를 변환해서 같은 위치에 두고 그 파일을 쓰게 해야 했다.

## 사용한 프롬프트

```text
각 캐릭터별 쓰이는 모델과 어디서 정보를 찾고 검색하는지, 어떤 프롬프트로 입력하는지, 결과값 표시는 어떻게 하는지 8명 다 정리해줘. 그리고 이 정리한걸 파일로 만들어서 그 파일 내용을 수정하면 거기에 맞게 프롬프트나 정보소스를 내가 직접 편집할수 있도록 해줘.

그리고 렛저webp 못찾겠으면 지금 png파일 변환해서 렛저webp로 만든후 다른 캐릭터들 있는곳에 넣어놓고 그 이미지 불러오게 해
```

## 해결된 것

- 8명 설정을 단일 TS 파일로 모으고, 실제 코드가 그 파일을 보도록 연결했다.
- 사람이 읽는 참조 문서도 같이 추가했다.
- `ledger.webp`를 새로 생성하고 참조를 다시 `webp`로 맞췄다.

## 아직 안 된 것

- 설정 파일 변경 이력을 UI에서 직접 diff로 보여주거나 자동 생성하는 기능은 추가하지 않았다.
