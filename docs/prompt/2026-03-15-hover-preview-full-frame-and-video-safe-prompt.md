# hover 프리뷰 전신 노출과 비디오 안전 프레임 조정 프롬프트

- 작성시각: 2026-03-15 15:42 KST
- 해결하고자 한 문제:
  - hover 프리뷰를 더 크게 보여주고, mp4를 넣었을 때도 캐릭터가 다 보이게 해야 함
- 사용한 프롬프트:
  - 호버프리뷰도 같은 방식으로 캐릭터 이미지로 배경 꽉 채워줘. 이번엔 호버프리뷰 크기를 늘려도 되니 캐릭터가 다 100% 나오게끔 해줘. 그래야 mp4영상 넣었을떄 안짤린다
- 해결된 것:
  - hover 프리뷰를 더 크게 조정함
  - 비디오와 이미지 모두 전신이 보이게 `contain` 기준으로 정리함
  - 배경도 캐릭터 이미지 기반으로 채워 프리뷰가 꽉 차 보이게 함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 실제 mp4 파일은 아직 없음
