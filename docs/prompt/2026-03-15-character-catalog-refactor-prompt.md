# 캐릭터 데이터 편집 구조 분리 프롬프트

- 작성시각: 2026-03-15 11:55 KST
- 해결하고자 한 문제: 캐릭터를 더 추가하거나 내용과 이미지를 나중에 바꿀 때 한 파일만 수정하면 되게 구조를 정리하고 싶었어.
- 사용한 프롬프트:
  - 나중에 캐릭터를 더 추가하거나 캐릭터 성격이나 내용을 변경할거를 대비해서 확장하거나 수정하기 쉽게 관련 파일을 따로 콤포넌트로 만들어서 편집하기 쉽게 구성해줘. 그래서 그 파일만 수정하면 이미지나 관련 내용이나 실데이터api를 변경하기 편하도록
- 해결된 것:
  - 단일 편집 파일과 파생 데이터 구조를 만들었어.
  - 공통 이미지 컴포넌트를 추가했어.
  - API 확장을 위한 seed 함수까지 준비했어.
- 해결되지 않은 것:
