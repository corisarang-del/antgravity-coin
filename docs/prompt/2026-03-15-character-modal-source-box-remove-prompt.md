# 캐릭터 모달 소스 박스 제거 프롬프트

- 작성시각: 2026-03-15 15:17 KST
- 해결하고자 한 문제:
  - 캐릭터도감 모달 하단의 원본 이미지/배포 이미지 소스 박스를 없애고 프레임도 더 짧게 정리해야 했음
- 사용한 프롬프트:
  - 캐릭터도감에서 클릭하면 나오는 모달에서 밑에 원본이미지, 배포이미지 소스부분 없애도 된다. 그리고 프레임을 없어진 부분만큼 줄여줘
- 해결된 것:
  - 하단 소스 박스를 제거함
  - 모달 높이를 더 타이트하게 정리함
  - 테스트까지 통과함
- 해결되지 않은 것:
  - 없음
