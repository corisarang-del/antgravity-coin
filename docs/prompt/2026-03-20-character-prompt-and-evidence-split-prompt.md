# Character Prompt And Evidence Split Prompt

- 작성시각: 2026-03-20 20:45 KST
- 프롬프트 요약:
  - 캐릭터 배틀 토론이 전부 비슷한 차트형 답변으로 수렴하는 문제를 개선
  - 프롬프트와 근거 분배를 다시 설계해서 역할 차이를 강화
  - 원소스가 보이게 하고 캐시도 새 프롬프트 기준으로 갱신되게 만들기

## 해결된 것

- 프롬프트 구조와 근거 문자열을 역할 중심으로 재설계했음
- 캐시 버전을 올려 이전 프롬프트 결과 재사용을 막았음
- 정적 검증과 전체 테스트를 통과했음

## 해결되지 않은 것

- 비차트 캐릭터를 더 살리려면 현재 숫자형 데이터 외에 뉴스 원문/이벤트 설명 입력이 추가로 필요함
