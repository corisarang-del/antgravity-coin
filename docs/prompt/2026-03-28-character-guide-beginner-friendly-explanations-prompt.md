# 캐릭터 도감 초보용 해설 추가 프롬프트

- 작성시각: 2026-03-28 21:08 KST

## 프롬프트 요약

- 캐릭터 도감에서 어려운 내용들. 예를들면 RSI나 이런 지표들을 초보들도 알기쉽게 해설같은걸 달던지 해서 더 이해하기 쉽게 바꿨으면 좋겠어
- 그전에 있던 캐릭터 성격이 완전히 없어져버렸다. 그거 다시 복구시켜주고 초보용 해설은 1-2개 어려운 부분만 넣어줘. 일일히 다 넣을 필요는 없다
- 캐릭터도감에서 고른 이유는 그냥 빼줘. 그리고 관련 문서들 프로젝트 소스코드에 맞게 추가로 업데이트 해주고 커밋 및 푸시해줘

## 해결하고자 한 문제

- 도감 안의 전문 용어를 초보도 직관적으로 이해할 수 있게 만들고, 카드와 모달 모두에서 설명 밀도를 다르게 제공하는 것

## 해결된 것

- 카드에는 쉬운 한줄 설명을 넣고, 모달에는 용어별 자세한 해설을 넣는 방향으로 구현했음
- 8명 전 캐릭터에 대해 공통 구조의 초보용 해설 데이터를 추가했음
- 추가 피드백 반영으로 성격/고른 이유 노출 순서를 복구하고, 초보 해설은 캐릭터당 1~2개 어려운 포인트만 남기도록 축소했음
- 추가 피드백 반영으로 캐릭터 도감 모달에서는 `고른 이유`를 완전히 제거하고, 소개 문구와 planning 문서도 현재 UI 기준으로 정리했음

## 아직 안 된 것

- 배틀 화면과 랜딩 캐릭터 레일 카피까지는 이번 수정 범위에 포함하지 않았음
