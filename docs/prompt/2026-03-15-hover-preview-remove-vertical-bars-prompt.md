# hover 프리뷰 상하 여백 제거 프롬프트

- 작성시각: 2026-03-15 16:08 KST
- 해결하고자 한 문제:
  - hover 프리뷰 커지는 부분의 상하 배경 여백을 없애야 했음
- 사용한 프롬프트:
  - 업로딩한 이미지에서 상하부분 배경 이거 말하는거야. 랜딩페이지 보여지는 부분말고 마우스 포인트 갔다댔을떄 커지는 부분
- 해결된 것:
  - hover 프리뷰 미디어 프레임을 원본에 더 가까운 가로형으로 조정함
  - 내부 패딩을 제거해 상하 여백을 줄임
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 실제 mp4 원본별 세부 비율 차이는 아직 고려하지 않음
