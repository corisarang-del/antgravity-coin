# 부분 실데이터 실패 시 배틀 계속 진행 프롬프트

- 작성시각: 2026-03-17 17:02 KST
- 해결하고자 한 문제:
  - 실데이터 확보 실패가 배틀 전체 중단으로 이어지지 않게 하고, 해당 팀원만 사과 메시지를 출력하게 해야 했다.

## 사용한 프롬프트

```text
실데이터 확보 실패 시 배틀 중단이 아니고 안나온 팀원 메세지는 지금 의견을 제시하지 못해서 죄송합니다. 다음번에 하겠습니다 하고 메시지 출력. 그리고 배틀은 남은 사람들끼리 진행한다
```

## 해결된 것

- 보조 근거 소스 실패는 null로 처리하고 배틀은 계속 진행하게 했다.
- 필요한 근거가 비는 캐릭터는 사과 메시지를 출력하게 했다.

## 아직 안 된 것

- 핵심 시세(CoinGecko) 실패까지 partial 진행으로 바꾸지는 않았다.
