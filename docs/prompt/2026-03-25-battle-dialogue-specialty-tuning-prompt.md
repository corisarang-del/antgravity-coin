# 토론 발언 직전 맥락 노출 제거와 전문 분야 강화 프롬프트 기록

- 작성시각: 2026-03-25 14:47 KST
- 해결하고자 한 문제:
  - 캐릭터 토론 발언 대화창에서 직전 관련 내용 노출을 줄이고, 캐릭터별 맡은 분야가 발언에 더 잘 드러나게 만들어야 했음.

## 사용한 프롬프트

```text
캐릭터 토론 발언 대화창에서 직전 관련 내용은 표시 안되게 해주고 캐릭터가 맡은 분야가 더 잘 나타내도록 토론발언 내용 만들어줘 예를들어 judy같은 경우엔 뉴스나 공시, 정책일정. vela, ledger, clover도 마찬가지.
```

## 해결된 것

- 이전 발언을 내부 참고용으로만 다루도록 prompt 규칙 수정
- fallback 발언에서 직전 발언 직접 언급 제거
- Judy, Clover, Ledger, Vela 전문 분야 표현 강화
- 대상 파일 `eslint`, `typecheck` 확인

## 아직 안 된 것

- Vitest는 환경상 `spawn EPERM`으로 실행하지 못했음.
- 전체 `pnpm.cmd lint` 는 `tmp/` 산출물 문제로 실패했음.
