# 캐릭터 모달 단일 원본 이미지 꽉 채우기 프롬프트

- 작성시각: 2026-03-15 16:32 KST
- 해결하고자 한 문제:
  - 캐릭터도감 모달 상단은 배경 이미지 없이 원본 이미지 하나만으로 꽉 차게 보여야 했음
- 사용한 프롬프트:
  - 캐릭터도감에서 클릭하면 나오는 모달에서 배경을 이미지로 채우지말고 그냥 원본이미지로 꽉 채워서 보여줘. 배경에 이미지 넣으면 가독성이 떨어진다
- 해결된 것:
  - 배경 이미지 레이어를 제거함
  - 모달 상단을 단일 원본 이미지 `object-cover` 방식으로 전환함
  - `pnpm lint`, `pnpm typecheck`, `pnpm test -- CharacterDetailModal` 통과
- 해결되지 않은 것:
  - 없음
