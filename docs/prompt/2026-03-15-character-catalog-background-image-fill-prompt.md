# 캐릭터도감 배경까지 캐릭터 이미지로 채우기 프롬프트

- 작성시각: 2026-03-15 15:10 KST
- 해결하고자 한 문제:
  - 배경까지 캐릭터 이미지로 채우되 어디도 잘리지 않는 느낌으로 보여야 함
- 사용한 프롬프트:
  - 배경까지 다채워달란거였어. 배경까지 캐릭터이미지로 다 안짤리게 채워줘.
- 해결된 것:
  - 카드와 모달 모두 좌우 배경을 같은 캐릭터 이미지 복제로 채움
  - 중앙 캐릭터는 비율 유지로 전부 보이게 유지함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 캐릭터별 수동 오프셋 튜닝은 아직 하지 않음
