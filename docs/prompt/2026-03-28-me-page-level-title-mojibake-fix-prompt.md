# 내 페이지 등급명 깨짐 복구 프롬프트

- 작성시각: 2026-03-28 21:08 KST

## 프롬프트 요약

- 로그인해서 내페이지에 현재등급이 업로딩한 이미지처럼 깨져서 나온다

## 해결하고자 한 문제

- `/me` 화면의 `현재 등급` 문자열이 깨져 보이는 원인이 소스 문자열인지, 저장된 진행도 데이터인지 구분하고 실제 사용자 화면이 바로 복구되게 만드는 것

## 해결된 것

- 등급 title 생성 경로와 읽기 경로를 같이 점검하게 됨
- 기존 저장 데이터까지 복구할 수 있게 정규화 방식으로 수정 방향을 잡음
