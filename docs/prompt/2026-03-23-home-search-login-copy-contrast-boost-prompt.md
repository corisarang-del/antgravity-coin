# 홈 검색 로그인 설명문 대비 강화 프롬프트

- 작성시각: 2026-03-23 22:39 KST
- 프롬프트 요약:
  - 홈/검색/로그인 화면 기준으로 설명문 대비 색도 한 단계 올려서 더 또렷하게 맞춰줘

## 해결하고자 한 문제

- 홈, 검색, 로그인 설명문 가독성을 색 대비 측면에서도 조금 더 올리고 싶었음
- 전역 muted 톤을 깨지 않고 선택된 화면만 조정해야 했음

## 해결된 것

- 설명문 전용 대비 강화 클래스를 만들고 홈, 검색, 로그인에만 적용했음
- `typecheck`와 대상 파일 `eslint`로 확인했음

## 해결되지 않은 것

- 실제 브라우저 시각 검수는 따로 하지 않았음
