# live battle 이후 선택/결과 화면 테스트 프롬프트

작성시각: 2026-03-21 KST

## 해결하려고 한 문제

- 캐릭터 토론 후 실제 선택 화면, 결과 승패까지 제대로 되는지 테스트

## 사용한 프롬프트 요약

- `캐릭터 토론후 실제 선택화면, 결과 승패까지 제대로 되는지 테스트`
- live 브라우저 기준으로 홈 -> 배틀 -> 선택 -> waiting -> 결과 흐름을 가능한 범위까지 직접 재현해줘
- live 토론이 너무 느리면 snapshot/localStorage와 기존 outcome seed를 써서 선택/결과 화면 기능도 분리 검증해줘

## 해결된 것

- 선택 화면 동작 확인
- waiting 화면 동작 확인
- 기존 outcome seed를 통한 결과 화면 렌더링 확인

## 아직 안 된 것
