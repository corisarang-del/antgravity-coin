# 헤더 홈 진입점을 랜딩으로 바꾸는 프롬프트

- 작성시각: 2026-03-28 21:08 KST

## 프롬프트 요약

- 그리고 상단에 홈 버튼이랑 좌측상단 AntGravity 로고 눌렀을때 배틀화면이 아닌 트레이더 시즌1 라인업이 있는 그 화면으로 가고 싶다. 지금은 홈 버튼이나 로고 누르면 배틀화면으로 가서 유저 흐름이 이상하다

## 해결하고자 한 문제

- 헤더의 주요 홈 진입점이 사용자가 기대하는 랜딩이 아니라 배틀 화면으로 연결돼 있는 문제

## 해결된 것

- AppHeader 기준 홈 링크와 로고 링크를 모두 랜딩 `/`로 맞추는 방향으로 수정했음
