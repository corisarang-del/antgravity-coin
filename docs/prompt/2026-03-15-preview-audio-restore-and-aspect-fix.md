# 프리뷰 오디오 복구와 비디오 비율 복원

- 작성시각: 2026-03-15 23:15:08 +09:00
- 해결하고자 한 문제:
  - 프리뷰 호버 비디오에 상하 검은 배경이 남아 있었음.
  - 이전 변환에서 오디오가 제거돼 프리뷰 영상 소리가 나지 않았음.
- 사용한 프롬프트/지시:
  - 검은 배경이 사라지도록 프리뷰 비디오 비율을 복원한다.
  - ffmpeg로 다시 변환해 오디오가 포함되게 한다.
- 해결된 것:
  - `design/character/*.mp4` 원본에서 다시 변환해 `public/characters/previews/*.mp4`를 덮어씀.
  - 변환 결과는 `720x396` 원본 비율 유지 + `aac` 오디오 포함 + `+faststart` 기준으로 맞춤.
  - 출력 파일에 `aac` 오디오 트랙이 실제로 포함된 걸 확인함.
  - 프리뷰 레이아웃은 `aspect-[20/11]` 기준으로 맞춰 상하 검은 배경이 줄어들게 구성함.
- 해결되지 않은 것:
  - 브라우저 자동재생 정책 때문에 첫 사용자 인터랙션 전에는 hover 사운드가 차단될 수 있음.
