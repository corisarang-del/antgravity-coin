# 캐릭터 저장소 스위칭 구조 추가 프롬프트

- 작성시각: 2026-03-15 12:28 KST
- 해결하고자 한 문제: `/api/characters`를 실제 저장소 인터페이스로 감싸고, 로컬 카탈로그와 외부 API를 환경 설정으로 스위칭되게 만들고 싶었어.
- 사용한 프롬프트:
  - /api/characters를 실제 저장소 인터페이스로 감싸서, 로컬 카탈로그와 외부 API를 환경 설정으로 스위칭되게까지 이어서 정리해줘

  - 저장소 포트, 유스케이스, 인프라 구현, 팩토리, 환경 설정, 테스트를 전부 반영했어.
- 해결되지 않은 것:
  - 실제 외부 운영 API 주소는 아직 비어 있어.
