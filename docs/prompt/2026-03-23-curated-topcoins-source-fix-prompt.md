# 추천 코인 curated 소스 경로 수정 프롬프트

- 작성시각: 2026-03-23 23:18 KST
- 프롬프트 요약:
  - avax로 교체했다는데 여전히 FIGR_HELOC 코인이 나온다.

## 해결하고자 한 문제

- 추천 코인 상수는 바뀌었는데 실제 홈 화면 목록은 그대로라서, 진짜 데이터 소스 경로를 수정해야 했음

## 해결된 것

- `fetchTopCoins()`를 curated coin id 기반으로 바꿔 홈 추천 코인 목록이 현재 소스를 따르게 했음
- `typecheck`와 대상 파일 `eslint`를 확인했음

## 해결되지 않은 것

- 이미 떠 있는 dev 서버나 캐시 화면은 새 코드 반영 전까지 이전 목록이 보일 수 있음
