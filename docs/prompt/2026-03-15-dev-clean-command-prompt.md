# 개발서버 포트 정리 명령 추가 프롬프트

- 작성시각: 2026-03-15 10:40 KST
- 해결하고자 한 문제: 현재 띄어져 있는 개발 서버 포트를 종료하고, 다음부터 재사용 가능한 자동 정리 명령을 만들고 싶었음.
- 사용한 프롬프트:
  - 포트 점유 프로세스 자동 정리용 명령 만들어주고 현재 띄어져있는 개발서버 포트 죽여줘
- 해결된 것:
  - 현재 `3001` 포트를 점유한 개발 서버를 종료함.
  - `pnpm run dev:clean` 명령을 만들었음.
- 해결되지 않은 것:
  - 개발 서버 재시작 자체는 사용자가 다시 실행해야 함.
