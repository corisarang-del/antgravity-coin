# hover 프리뷰 정사각 미디어 프레임 적용 프롬프트

- 작성시각: 2026-03-15 16:15 KST
- 해결하고자 한 문제:
  - hover 프리뷰 안의 좌우 검은 여백을 완전히 없애야 했음
- 사용한 프롬프트:
  - hover 프리뷰 안에서 좌우 검은 여백도 없애줘.
  - 업로딩한 이미지 보면 여전히 좌우 검은 여백이 보인다. 덜 보이는 수준말고 아예 안보이게 해줘
- 해결된 것:
  - 캐릭터 원본이 정사각형임을 확인함
  - hover 프리뷰 미디어 프레임을 정사각형으로 조정함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
