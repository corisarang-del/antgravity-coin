# 캐릭터 이미지 비율 조정과 프리뷰 오버레이 제거 프롬프트

- 작성시각: 2026-03-15 14:35 KST
- 해결하고자 한 문제:
  - 개발 서버 기준으로 캐릭터 도감과 설명 모달에서 캐릭터 이미지가 잘리지 않게 보여야 함
  - 랜딩 캐릭터 라인업에서 업로드 이미지 기준으로 거슬리는 프리뷰 오버레이를 제거해야 함
- 사용한 프롬프트:
  - 현재 배포할때 말고 개발서버에서도 캐릭터 이미지가 webp로 되있는건가?
  - 그럼 원본 이미지 파일을 내가 지워도 될까? 그리고 캐릭터도감에서 이미지가 100% 다 나오게끔 비율조정해줘. 클릭했을때 나오는 설명모달에도 캐릭터 이미지가 다 보이도록 비율조정. 그리고 업로딩한 이미지에서 저 부분은 다 빼줘
- 해결된 것:
  - 런타임에서 실제로 쓰는 캐릭터 파일이 `public/characters/*.webp`라는 점을 확인함
  - 도감 카드, 설명 모달, 랜딩 클릭 모달, 랜딩 포스터 카드의 이미지 비율을 `contain` 중심으로 조정함
  - 랜딩 hover 프리뷰 오버레이를 제거함
  - `pnpm lint`, `pnpm typecheck`, `pnpm test -- CharacterDetailModal`까지 확인함
- 해결되지 않은 것:
  - 원본 PNG 파일 삭제는 아직 수행하지 않았고, 삭제 범위는 사용자 확인 후 진행하는 편이 안전함
