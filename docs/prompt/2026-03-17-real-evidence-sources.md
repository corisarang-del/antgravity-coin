# 실근거 소스 강제 및 외부 API 실호출 전환 프롬프트

- 작성시각: 2026-03-17 14:55 KST
- 해결하고자 한 문제:
  - 문서와 코드 모두에서 근거 소스를 실호출/실데이터 기준으로 맞춰야 했다.
  - `CryptoPanic`, `Coinglass`, `mock/synthetic` 경로를 실제 데이터 파이프라인으로 교체해야 했다.

## 사용한 프롬프트

```text
08-character-debate-reference.md에 각 근거 소스를 실호출만 되게끔 명확히 표기. CryptoPanic, Coinglass를 실제 API 호출로 바꾸기. 그외에 다른 근거소스들도 실제 api호출이나 데이터로만 얻어오게 해줘
```

## 해결된 것

- 외부 API 클라이언트를 실호출 기준으로 바꾸고, `fetchMarketData`에서 실데이터만 사용하도록 정리했다.
- 근거 소스 문서도 `실호출`과 `실데이터 기반 계산`으로 명확히 바꿨다.

## 아직 안 된 것

- 실제 배틀 실행에는 관련 API 키가 준비돼 있어야 한다.
