# 랜딩 라인업 소개영역 복구와 캐릭터 확대 프롬프트

- 작성시각: 2026-03-15 15:30 KST
- 해결하고자 한 문제:
  - 랜딩 시즌 라인업 카드 하단 소개영역은 유지해야 하고, 캐릭터는 더 크게 보여야 함
- 사용한 프롬프트:
  - 밑에있는 캐릭터 소개란은 어디로 사라졌지. 그건 그대로 내비두고 컬러배경판을 캐릭터이미지에 맞추란거였어. 근데 캐릭터 이미지가 너무 작게 축소됬다. 더 키워줘
- 해결된 것:
  - 하단 소개영역을 복구함
  - 캐릭터 이미지를 더 키움
  - 컬러 배경판은 캐릭터 폭 기준으로 더 타이트하게 조정함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 캐릭터별 세부 배율은 아직 공통값으로 동작함
