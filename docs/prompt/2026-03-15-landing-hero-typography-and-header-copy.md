# 랜딩 히어로 타이포 정리와 상단 카피 변경

- 작성시각: 2026-03-15 21:58:57 +09:00
- 해결하고자 한 문제:
  - 랜딩 히어로의 `코인분석을 캐릭터 토론으로` 타이틀이 줄 간격과 자간이 어색해 보였음.
  - 좌상단 서브카피를 `코인 시즌 트레이더 배틀`로 바꾸고 싶었음.
- 사용한 프롬프트/지시:
  - 히어로 타이틀의 높낮이와 글자 사이 간격을 확인하고 어색하면 수정한다.
  - 좌상단 `코인 시즌 배틀 캐릭터 데이터`를 `코인 시즌 트레이더 배틀`로 변경한다.
  - 브라우저로 랜딩 페이지를 다시 확인하고 lint/typecheck를 돌린다.
- 해결된 것:
  - 히어로 타이틀을 두 줄 `span` 구조로 정리해 줄 간격과 자간을 안정화함.
  - 좌상단 서브카피를 `코인 시즌 트레이더 배틀`로 변경함.
  - 랜딩 파일에 남아 있던 깨진 구문과 문자열도 함께 정상화함.
- 해결되지 않은 것:
  - 랜딩 페이지의 다른 카피 톤 조정은 이번 요청 범위에서 더 건드리지 않았음.
