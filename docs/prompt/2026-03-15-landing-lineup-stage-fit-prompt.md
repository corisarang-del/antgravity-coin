# 랜딩 시즌 라인업 배경판 축소 정렬 프롬프트

- 작성시각: 2026-03-15 15:22 KST
- 해결하고자 한 문제:
  - 랜딩 시즌 라인업 카드에서 배경을 캐릭터 이미지로 채우지 않고, 캐릭터 크기에 맞게 배경판을 줄이는 방식으로 더 꽉 차 보이게 해야 함
- 사용한 프롬프트:
  - 배경 캐릭터 이미지로 채우지말고 차라리 캐릭터 이미지크기에 맞게 배경을 줄이는식으로 꽉채워줘. 모든 캐릭터 다 같이 수정. 디자인규칙 절대 지키면서.
- 해결된 것:
  - 캐릭터 복제 배경을 제거함
  - 카드 내부 배경판을 캐릭터 폭 중심으로 줄여 디자인 톤을 유지하면서 빈 영역 체감을 줄임
  - 캐릭터 본체는 높이를 더 쓰게 조정함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 캐릭터별 개별 위치 튜닝은 필요 시 추가 가능
