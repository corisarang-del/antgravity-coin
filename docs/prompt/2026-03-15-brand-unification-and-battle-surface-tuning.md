# 브랜드 통일과 배틀 화면 표면 레이어 조정

- 작성시각: 2026-03-15 22:27:02 +09:00
- 해결하고자 한 문제:
  - 랜딩과 내부 페이지의 브랜드 톤이 나뉘어 보여 일관성이 깨져 있었음.
  - `/battle/[coinId]` 화면의 표면 대비가 디자인 기준보다 평평해 보여 색감/레이어만 조정할 필요가 있었음.
  - 랜딩 히어로 문구를 `캐릭터토론으로`로 붙여 타이포 밀도를 맞추고 싶었음.
- 사용한 프롬프트/지시:
  - 브랜드 톤을 랜딩 기준으로 내부 페이지까지 일원화한다.
  - `/battle/[coinId]`는 크기 변경 없이 표면 대비만 올린다.
  - 랜딩 히어로 문구를 `캐릭터토론으로`로 변경한다.
  - 브라우저 확인과 `pnpm test`, `pnpm lint`, `pnpm typecheck`를 실행한다.
- 해결된 것:
  - 내부 헤더와 메타데이터를 `Ant Gravity` / `코인 시즌 트레이더 배틀` 기준으로 통일함.
  - 랜딩 히어로를 `코인분석을 / 캐릭터토론으로`로 정리함.
  - 배틀 메인 화면의 상단 카드, 팀 보드, 스포트라이트, 지표 카드, 피드 표면 레이어를 강화함.
  - 브라우저에서 랜딩과 배틀 화면을 다시 확인함.
- 해결되지 않은 것:
  - favicon 404는 기존 개발 환경 이슈라 이번 수정 범위에서 다루지 않았음.
