# login page design refresh prompt

- 작성시각: 2026-03-23 21:48 KST
- 프롬프트 요약:
  - 로그인 페이지를 현재 디자인 규칙에 맞게 더 정돈하고 보기 좋게 다듬기
  - 헤더의 `로그인/회원가입` 버튼도 다른 상단 버튼들과 조화되게 조정하기

## 해결하고자 한 문제

- 로그인 페이지는 기능은 맞지만 시각적 계층이 평평해서 현재 프로젝트의 디자인 규칙 대비 임팩트가 약했음
- 상단 `로그인/회원가입` CTA는 다른 헤더 버튼군과 비교해 톤이 조금 분리돼 보여 조화가 덜했음

## 해결된 것

- 로그인 페이지를 좌측 소개 패널 + 우측 소셜 진입 패널 구조로 더 분명하게 재구성했음
- Google/Kakao 버튼을 공통 카드 스타일 안에서 브랜드 배지로만 차별화해서 전체 톤을 유지했음
- 헤더의 `로그인/회원가입` 버튼을 상단 버튼군과 어울리는 카드형 CTA로 조정했음

## 해결되지 않은 것

- 실제 브라우저에서 최종 spacing과 hover 체감은 추가 확인이 필요함
