# hover 프리뷰 프레임 축소 정리 프롬프트

- 작성시각: 2026-03-15 15:53 KST
- 해결하고자 한 문제:
  - hover 프리뷰의 남는 영역을 캐릭터 배경으로 채우지 말고 프레임 자체를 타이트하게 정리해야 했음
- 사용한 프롬프트:
  - 프리뷰 프레임의 어두운 그라데이션 안으로만 보이게 하지말고 프레임자체 길이를 줄여서 이미지로만 꽉 차게 보이게 수정
- 해결된 것:
  - hover 프리뷰에서 배경 캐릭터 이미지 레이어를 제거함
  - 비디오와 이미지 폴백은 안 잘리게 유지함
  - 남는 영역은 프레임 안의 단순한 어두운 톤으로만 처리함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 실제 mp4 적용 후 캐릭터별 세부 프레임 조정은 필요 시 추가 가능
