# 캐릭터도감 설명문 가독성 규칙 동기화 프롬프트

- 작성시각: 2026-03-23 22:54 KST
- 프롬프트 요약:
  - 캐릭터도감도 동일 규칙으로 맞춰줘

## 해결하고자 한 문제

- 캐릭터도감 설명문/보조 카피도 홈/검색/로그인과 같은 가독성 규칙으로 맞춰야 했음
- 설명문만 키우고, 역할 같은 메타 텍스트는 그대로 유지해야 했음

## 해결된 것

- 캐릭터도감 페이지, 카드, 상세 모달의 설명문에 동일한 `16px + 대비 강화` 규칙을 적용했음
- `typecheck`와 대상 파일 `eslint`로 확인했음

## 해결되지 않은 것

- 실제 브라우저 시각 검수는 따로 하지 않았음
