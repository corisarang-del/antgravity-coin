## 해결하려던 문제

- 첫 홈 화면 전에 넷플릭스형 랜딩 페이지 추가
- 참고 이미지 기준 색감 적용
- design 폴더 안 토큰과 MASTER.md 동기화

## 사용한 프롬프트

```text
이제 첫 홈화면전에 이 앤트그래비티의 첫 느낌을 살려줄 랜딩페이지를 만들려고한다. 랜딩페이지로 강렬한 이미지를 준후 지금의 홈화면으로 넘어가는 구조다. 렌딩페이지는 넷플릭스 구조로 가는데 디자인이나 색감은[Image #1] 이걸 참고로 한다. 이런 구조로 design 폴더안의 관련 토큰과 master.md 동기화 해줘. 참고로 공통컴퍼넌트만 디자인이나 색감을 가져가고 큰 형식은 넷플릭스랑 같다. 넷플릭스 참고용도 refer폴더안에 있다
```

## 해결된 것

- 랜딩 페이지 구현
- 기존 홈을 `/home`으로 이동
- 토큰과 design/MASTER.md 동기화
