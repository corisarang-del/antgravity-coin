# 발표용 PPTX 제작 프롬프트

- 작성시각: 2026-03-16 00:00:00 +09:00
- 해결하고자 한 문제:
  - 현재 프로젝트를 설명하는 발표용 PPTX 보고서가 필요했음.
  - 웹앱 목적과 캐릭터 이미지를 넣고, 홈페이지 디자인에 맞는 시각 스타일로 구성해야 했음.
- 사용한 프롬프트/지시:
  - `[$pptx-design-styles](C:\\Users\\khc\\.codex\\skills\\pptx-design-styles\\SKILL.md) 현재 이 프로젝트를 발표할려고 하는 pptx를 만들거야. 이 웹앱의 목적과 캐릭터들 이미지도 넣어서 발표용 보고서 만들어줘. 디자인은 홈페이지 디자인 참고해서 거기에 맞게끔 피피티 작성`
- 해결된 것:
  - 홈페이지 톤을 참고한 글래스형 발표 자료를 설계함.
  - 캐릭터 8인 이미지를 포함한 7장 분량의 PPTX를 생성함.
  - `.pptx`와 재생성용 `.js`를 함께 남김.
- 해결되지 않은 것:
  - 현재 환경에서는 렌더/폰트 자동 검증 스크립트를 끝까지 통과시키지 못했음.
