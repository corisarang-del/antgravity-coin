# 홈 카피 깨짐 회귀 수정

- 작성시각: 2026-03-15 21:34:03 +09:00
- 해결하고자 한 문제:
  - 코드 리뷰에서 `TopCoinsGrid.tsx`, `RecentCoinsList.tsx`의 한글 카피가 깨진 P1 회귀가 발견됨.
  - 사용자에게 바로 보이는 홈 화면 텍스트라서 즉시 복구와 회귀 방지 테스트가 필요했음.
- 사용한 프롬프트/지시:
  - 리뷰 findings 기준으로 두 파일의 한국어 카피를 복구한다.
  - `next/link` 변경은 유지하고, 텍스트 회귀를 막는 컴포넌트 테스트를 추가한다.
  - `pnpm test`, `pnpm lint`, `pnpm typecheck`로 검증한다.
- 해결된 것:
  - 상위 코인 카드의 `시가총액`, `배틀 입장` 문구를 복구함.
  - 최근 본 코인 섹션의 제목, 설명, `다시 보기` 문구를 복구함.
  - 두 컴포넌트에 정적 카피 회귀 테스트를 추가함.
- 해결되지 않은 것:
  - 다른 파일에 남아 있을 수 있는 깨진 한글 전반은 이번 수정 범위에서 다루지 않았음.
