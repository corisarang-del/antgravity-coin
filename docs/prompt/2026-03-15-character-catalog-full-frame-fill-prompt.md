# 캐릭터도감 이미지 프레임 꽉 채움 보정 프롬프트

- 작성시각: 2026-03-15 15:02 KST
- 해결하고자 한 문제:
  - 캐릭터도감에서 이미지 좌우 여백이 많아 보여 프레임이 꽉 차지 않는 느낌을 줄여야 함
- 사용한 프롬프트:
  - 여전히 캐릭터도감에서 이미지 좌우로 빈 부분이 많다. 완전히 꽉 채워줘
- 해결된 것:
  - 도감 카드와 설명 모달 모두 배경은 꽉 차고 실제 캐릭터는 비율 유지로 전부 보이는 이중 레이어 방식으로 조정함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 캐릭터별 미세 크롭 오프셋 조정은 추후 필요 시 별도로 가능
