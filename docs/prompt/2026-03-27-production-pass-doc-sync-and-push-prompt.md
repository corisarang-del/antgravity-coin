## 해결하고자 한 문제

- production 실제 흐름 통과를 확인한 뒤, 관련 기존 문서에 그 결과를 반영하고 마지막 커밋/푸시까지 이어가야 했음

## 해결된 것

- production `BTC -> pick -> waiting -> result` 흐름 통과를 확인했음
- 기존 문서에 구조 변경점과 실테스트 결과를 추가 반영했음
- `.vercelignore`를 추가해 실제 배포에 필요한 파일만 올라가게 정리했음

## 해결되지 않은 것

- 이번 기록은 production 통과와 문서 동기화까지만 포함했고, 추가 기능 변경은 포함하지 않았음
