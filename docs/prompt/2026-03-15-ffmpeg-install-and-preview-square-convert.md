# ffmpeg 설치와 프리뷰 mp4 정사각 변환

- 작성시각: 2026-03-15 23:07:41 +09:00
- 해결하고자 한 문제:
  - 프리뷰 호버 비디오가 정사각 프레임에서 위아래 안 잘리고 꽉 차게 보여야 했음.
  - raw mp4를 그대로 쓰는 대신 `ffmpeg`로 실제 변환을 적용해야 했음.
- 사용한 프롬프트/지시:
  - `ffmpeg`를 설치한 뒤 preview mp4를 변환한다.
  - 프리뷰 비디오가 정사각 프레임에 꽉 차 보이도록 자산과 프론트 표시 방식을 맞춘다.
- 해결된 것:
  - `tools/ffmpeg` 아래에 ffmpeg Windows 빌드를 설치함.
  - `public/characters/previews/*.mp4`를 720x720 정사각 패딩, 오디오 제거, `yuv420p`, `+faststart` 기준으로 재변환함.
  - 프론트 비디오 표시를 `object-cover`로 바꿔 정사각 변환본이 프레임을 꽉 채우게 함.
  - 변환 후 mp4 용량이 대체로 0.48MB~0.79MB 수준으로 줄어듦.
- 해결되지 않은 것:
  - 없음.
