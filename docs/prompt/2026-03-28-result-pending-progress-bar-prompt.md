# 결과 페이지 준비 진행도 시각화 프롬프트

- 작성시각: 2026-03-28 22:05 KST

## 프롬프트 요약

- 업로딩한 이미지에서 결과페이지 준비중인걸 나타내는 로딩바로 남은 시간을 시각적으로 표현하고 싶다

## 해결하고자 한 문제

- 결과 페이지 pending 화면에서 남은 시간을 숫자만이 아니라 전체 바와 단계별 바로 더 직관적으로 보여주는 것

## 해결된 것

- 결과 페이지 전용으로 전체 진행 바와 단계별 진행 바를 같이 넣었음
- 전체 바는 실제 `selectedAt -> settlementAt` 시간을 기준으로 계산하고, settlement 이후 단계는 `resolveStage` 기반으로 표현하게 했음

## 아직 안 된 것

- waiting 화면까지 같은 시각화를 확장하는 건 이번 범위에 포함하지 않았음
