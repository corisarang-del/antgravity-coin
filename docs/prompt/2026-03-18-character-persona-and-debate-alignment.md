# 캐릭터 성격과 토론 답변 정합성 보강 프롬프트

- 작성시각: 2026-03-18 22:09 KST
- 해결하고자 한 문제:
  - 캐릭터 성격과 원본 소스, 토론 답변이 서로 다르게 느껴지고, 답변이 지나치게 차트 위주라는 문제를 잡고 싶었다.

## 사용한 프롬프트

```text
커밋해줘. 그런데 캐릭터에 주어진 성격이랑 실데이터 가져오는 원본소스랑 분석해서 내놓은 토론 답변이랑 조금 다른거 같다. 캐릭터 성격이랑 토론답변이 맞게끔 해줘. 전부 차트위주로만 답변을 내놓는거 같으니 검증부탁
```

## 해결된 것
- 캐릭터 역할/성격/선정 이유를 실제 토론 프롬프트에 포함시켰고, 역할별 fallback도 분리했다.
- 캐릭터 설명 텍스트를 정제해서 UI와 토론 입력이 같은 페르소나를 바라보도록 맞췄다.

## 아직 해결 안 된 것
- 운영 로그 기준 실제 LLM 응답 편차는 한 번 더 모니터링하는 게 좋다.
