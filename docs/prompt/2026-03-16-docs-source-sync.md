# 문서와 실제 소스 동기화 프롬프트

- 작성시각: 2026-03-16 00:00:00 +09:00
- 해결하고자 한 문제:
  - 실제 구현된 소스와 문서 사이 차이가 커져서 기준 문서로 쓰기 어려웠음.
  - 소스는 유지하고 문서만 현재 프로젝트 상태에 맞게 정확히 맞출 필요가 있었음.
- 사용한 프롬프트/지시:
  - 실제 구현된 소스파일과 문서와의 차이점을 알려줘
  - 절대 소스코드는 변경하지말고 문서만 현재 프로젝트 소스에 맞게 동기화
- 해결된 것:
  - `docs/planning` 핵심 화면/API 문서를 현재 구현 기준으로 갱신함.
  - `specs` 라우트/타입/컴포넌트 계약을 현재 소스와 맞춤.
  - 더 이상 쓰지 않는 `specs/screens/analyze.yaml`을 제거함.
- 해결되지 않은 것:
  - 실제 UI 문자열 깨짐은 소스 수정이 필요한 영역이라 이번 범위에서 제외함.
