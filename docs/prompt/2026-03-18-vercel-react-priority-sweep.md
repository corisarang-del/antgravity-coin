# vercel react 우선순위 전반 최적화 프롬프트

- 작성시각: 2026-03-18 20:19 KST
- 해결하고자 한 문제:
  - `$vercel-react-best-practices` 스킬에서 나온 우선순위를 기준으로 현재 코드베이스를 전반적으로 정리하고 싶었다.

## 사용한 프롬프트

```text
$vercel-react-best-practices

나온 우선순위 전부 수정
```

## 해결된 것
- 우선순위가 높은 항목부터 실제 코드베이스에서 적용 가능한 지점만 골라 서버 캐시, 병렬화, 번들 분리, 리렌더 감소, 중복 요청 제거를 반영했다.

## 아직 해결 안 된 것
- 모든 세부 규칙을 기계적으로 전부 넣지는 않았고, 현재 코드 흐름에서 효과가 분명한 부분 위주로 먼저 정리했다.
