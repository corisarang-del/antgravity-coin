# 재사용 메모 저장 구조와 admin 메모 가시화 프롬프트

- 작성시각: 2026-03-18 22:28 KST
- 해결하고자 한 문제:
  - Gemini 리포트를 다음 토론에 재사용 가능한 메모로 저장하고, admin에서 직접 볼 수 있게 만들고 싶었다.

## 사용한 프롬프트

```text
Gemini 리포트 -> 재사용 메모 저장 구조 추가
토론 시작 전 최근 메모 retrieval 추가
캐릭터별 lesson 주입까지 연결
```

## 해결된 것
- 리포트와 별개로 재사용 메모를 저장하게 했고, 다음 토론 시작 전에 최근 메모를 retrieval해서 캐릭터별 lesson까지 주입하게 했다.
- admin에서 메모 내용을 바로 볼 수 있게 했다.

## 아직 해결 안 된 것
- lesson JSON 결과를 더 다듬는 프롬프트 튜닝은 후속 여지가 있다.
