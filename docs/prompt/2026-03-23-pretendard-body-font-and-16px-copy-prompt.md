# Pretendard body 폰트 전환과 16px 설명문 가독성 개선 프롬프트

- 작성시각: 2026-03-23 22:34 KST
- 프롬프트 요약:
  - Pretendard body 폰트 전환 + 설명문 16px 가독성 개선 계획 구현 요청

## 해결하고자 한 문제

- body 폰트를 더 깔끔하고 한글 가독성이 좋은 무료 폰트로 바꾸고 싶었음
- 전역 전체 확대가 아니라 설명문 계열만 `16px`로 키워서 읽기 편하게 만들고 싶었음

## 해결된 것

- body 폰트를 `Pretendard` 중심으로 재구성했음
- 공통 설명문 클래스 `.ag-body-copy`를 추가해 주요 설명문에 `16px` 규칙을 일관 적용했음
- 홈, 검색, 로그인, waiting, pick, result, 공통 설명형 컴포넌트까지 반영했음
- `typecheck`와 대상 파일 `eslint`로 기본 검증했음

## 해결되지 않은 것

- 실제 브라우저 시각 검수는 따로 하지 않았음
- 더 많은 화면으로 확대 적용할지는 추가 피드백에 따라 결정 가능함
