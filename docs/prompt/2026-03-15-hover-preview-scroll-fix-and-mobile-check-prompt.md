# hover 프리뷰 스크롤 종료와 모바일 점검 프롬프트

- 작성시각: 2026-03-15 17:12 KST
- 해결하고자 한 문제:
  - hover 프리뷰가 스크롤에 따라 내려오는 버그를 고치고 모바일 반응성도 확인해야 했음
- 사용한 프롬프트:
  - 랜딩페이지에서 호버프리뷰가 실행된 상태로 마우스 스크롤을 하면 없어지지 않고 따라 내려온다. 그리고 모바일 반응성도 체크해줘
- 해결된 것:
  - 스크롤/리사이즈 시 hover 프리뷰가 닫히도록 수정함
  - 터치 환경에서 hover 프리뷰를 비활성화함
  - 데스크톱과 모바일 뷰포트에서 실제 동작 확인함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 실제 mp4 에셋 부재로 비디오 요청 404는 남아 있음
