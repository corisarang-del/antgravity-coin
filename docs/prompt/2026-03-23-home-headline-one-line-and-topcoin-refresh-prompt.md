# 홈 헤드라인 한 줄 정리와 추천 코인 교체 프롬프트

- 작성시각: 2026-03-23 22:49 KST
- 프롬프트 요약:
  - 어떤 코인으로 붙어볼지 골라줘 도 한줄로 보이게끔 위치조정. 불리시팀 vs 베어리시팀 이것도 한줄로 보이게끔 내부 위치 조정. FIGR_HELOC 대신 다른 유명한코인으로 대체

## 해결하고자 한 문제

- 홈과 검색 영역의 큰 문구를 한 줄로 더 깔끔하게 보이게 해야 했음
- 추천 코인 목록에서 더 유명한 코인으로 교체하고 싶었음

## 해결된 것

- 홈과 검색 헤드라인의 줄바꿈 방지와 크기 조정을 반영했음
- 추천 코인 목록은 `AVAX` 기준으로 갱신했음
- `typecheck`와 대상 파일 `eslint`로 확인했음

## 해결되지 않은 것

- 실제 브라우저 시각 검수는 따로 하지 않았음
