# 홈 검색 카드 외곽 overflow 수정 프롬프트

- 작성시각: 2026-03-23 23:03 KST
- 프롬프트 요약:
  - 업로딩한 이미지 보면 어떤코인으로 붙을지 골라줘 이쪽 칸이 외곽으로 벗어나있다. 이걸 벗어나지 않고 내부에 위치하도록 해줘

## 해결하고자 한 문제

- 홈 상단 오른쪽 검색 카드가 외곽 박스 밖으로 밀려 보이는 레이아웃 문제를 해결해야 했음
- 한 줄 헤드라인은 유지하되 컨테이너 안쪽에서만 자연스럽게 줄어들도록 조정해야 했음

## 해결된 것

- 홈 grid 최소 폭 규칙과 좌우 칸의 `min-w-0`를 조정했음
- 홈/검색 헤드라인 크기를 조금 더 보수적으로 다듬었음
- `typecheck`와 대상 파일 `eslint`를 확인했음

## 해결되지 않은 것

- 실제 브라우저 시각 검수는 따로 하지 않았음
