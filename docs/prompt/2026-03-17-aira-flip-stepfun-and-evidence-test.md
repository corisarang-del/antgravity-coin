# Aira/Flip 모델 교체와 근거 소스 검증 프롬프트

- 작성시각: 2026-03-17 14:34 KST
- 해결하고자 한 문제:
  - `Aira`, `Flip` 기본 모델을 바꾸고, 캐릭터 프로필에 적힌 근거 소스가 실제 데이터로 채워지는지 검증해야 했다.

## 사용한 프롬프트

```text
현재 flip이랑 aira가 qwen3.5로 되있는데 Step 3.5 Flash (free)로 둘다 바꿔주고 폴백시에만 qwen3.5 그리고 모든 캐릭터들이 적혀있는 근거소스에서 실제로 데이터를 얻어오는지 테스트해줘.
```

## 해결된 것

- `Aira`, `Flip` 기본 모델을 `Step 3.5 Flash (free)`로 바꾸고 fallback은 `Qwen3.5`로 유지했다.
- 프로필에 적힌 `market` 근거 소스들이 실제 `MarketData`에 채워지는지 테스트를 추가했다.

## 아직 안 된 것

- `previous_messages`는 시장 데이터가 아니라 대화 이력이라 이번 테스트에는 포함하지 않았다.
