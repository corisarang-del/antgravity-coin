# hover 프리뷰 복구와 비디오 폴백 정리 프롬프트

- 작성시각: 2026-03-15 14:47 KST
- 해결하고자 한 문제:
  - 첫 화면 핵심인 hover 캐릭터 소개 영상을 다시 보이게 해야 함
  - `preview.mp4` 같은 파일명 문구나 깨진 영역은 보이지 않게 해야 함
- 사용한 프롬프트:
  - 화면을 덮던 hover 프리뷰 오버레이는 원래대로 복구시켜줘. 그게 첫화면에서 캐릭터 소개영상 mp4로 보여주는 핵심임. 그리고 업로드한 이미지의 저 파일 나오는 부분 안보이게 해줘
- 해결된 것:
  - hover 프리뷰를 복구함
  - 실제 mp4가 있을 때 자동재생 가능한 구조로 정리함
  - 영상이 없을 때는 파일명이나 에러 대신 이미지 프리뷰로 폴백되게 바꿈
  - `pnpm lint`, `pnpm typecheck`를 통과함
- 해결되지 않은 것:
  - 각 캐릭터별 실제 mp4 에셋은 아직 추가되지 않음
