# 캐시 전략 문서화 및 구현 프롬프트

- 작성시각: 2026-03-18 19:55 KST
- 해결하고자 한 문제:
  - 무료 호출량 안에서 운영 가능한 캐시 스키마, 배치 수집 주기, stale 정책을 정리하고 그 기준으로 바로 구현해야 했다.

## 사용한 프롬프트

```text
캐시 JSON 스키마
무료 호출량 기준 배치 수집 주기표
화면별 stale 데이터 허용 규칙
그다음 그 문서 기준으로 바로 구현
```

## 해결된 것

- 캐시 전략 문서를 추가했고, 파일 캐시 저장소와 prewarm 진입점까지 구현했다.

## 아직 안 된 것

- 외부 주기 실행기 자체는 아직 연결하지 않았다.
