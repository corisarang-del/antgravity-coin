# vercel react 저우선순위 2차 스윕 프롬프트

- 작성시각: 2026-03-18 20:29 KST
- 해결하고자 한 문제:
  - 1차 정리 후 남은 low priority 항목도 가능한 범위까지 계속 밀어붙여 정리하고 싶었다.

## 사용한 프롬프트

```text
저우선순위 항목까지 더 밀어붙여서 2차 스윕하고 싶으면 이어서 남은 후보도 끝까지 정리
```

## 해결된 것
- low priority 범주에서 실익이 있는 후보를 골라 외부 스토어, 이벤트 핸들러, 배열 순회, 렌더링 미세 최적화를 반영했다.

## 아직 해결 안 된 것
- 실측 병목이 없는 항목까지 전부 기계적으로 적용하지는 않았다.
