# 로그인 카피 축소와 카드 정렬 조정

- 작성시각: 2026-03-23 22:19 KST
- 해결하고자 한 문제:
  - 로그인 헤드라인을 더 짧게 줄이고 싶었음
  - Google, Kakao 로그인 카드의 보조 설명 문구가 남아 있어 여전히 정보량이 많아 보였음

## 해결된 것

- `src/app/login/LoginPageClient.tsx`의 메인 헤드라인을 `바로시작`으로 변경했음
- Google, Kakao 카드의 보조 설명 문구를 제거했음
- 설명 문구를 뺀 뒤 버튼 내부 높이와 정렬이 어색하지 않게 `min-h-11` 기준으로 재정렬했음
- 검증
  - `node_modules\.bin\eslint.cmd src/app/login/LoginPageClient.tsx` 통과

## 해결되지 않은 것

- 실제 브라우저에서 최종 시각 확인은 이번 턴에서 별도로 하지 않았음
- 로그인 페이지의 나머지 보조 문구를 더 줄일지는 다음 피드백에 따라 조정 가능함
