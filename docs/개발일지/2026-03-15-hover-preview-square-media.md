# hover 프리뷰 정사각 미디어 프레임 적용

- 작성시각: 2026-03-15 16:15 KST
- 해결하고자 한 문제:
  - hover 프리뷰 안에서 좌우 검은 여백이 여전히 보였음
  - 실제 캐릭터 이미지 원본 비율에 맞춰 프레임을 조정해야 했음
- 해결된 것:
  - `public/characters/*.webp` 원본이 모두 `1024x1024` 정사각형인 점을 확인함
  - hover 프리뷰 미디어 영역을 정사각형으로 조정함
  - 프리뷰 폭과 위치를 함께 보정해 여백 없이 더 자연스럽게 보이게 맞춤
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 향후 mp4가 정사각형이 아니면 비율에 따라 추가 조정이 필요할 수 있음
