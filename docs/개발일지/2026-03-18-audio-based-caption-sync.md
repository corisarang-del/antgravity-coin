# 오디오 기준 프레임 자막 싱크 적용

- 작성시각: 2026-03-18 23:50 KST
- 해결하고자 한 문제: 내레이션 오디오가 들어간 뒤에도 자막은 장면 단위로만 고정돼 있어서 실제 발화 타이밍과 맞지 않았다. 문장별 오디오 길이를 기준으로 프레임 단위 자막 싱크를 맞추고 싶었다.

## 이번 단계에서 해결한 것

- Gemini TTS를 줄 단위로 따로 생성하고 하나의 내레이션 트랙으로 합치는 방식으로 바꿨다.
- 생성 과정에서 문장별 시작/종료 시각을 JSON으로 저장하도록 했다.
- Remotion에서 해당 JSON을 읽어 현재 프레임 기준으로 자막을 표시하도록 변경했다.
- 가로판과 확장판은 목표 영상 길이에 맞춰 오디오와 타이밍 JSON을 같은 비율로 압축했다.
- 세 가지 영상 산출물을 자막 싱크 반영본으로 다시 렌더했다.

## 아직 해결되지 않은 것

- 장면 전환과 자막 위치를 더 정교하게 연결하는 디자인 튜닝은 추가 여지가 있다.
- 음성 파형 기반 자동 줄바꿈이나 단어 단위 하이라이트는 아직 적용하지 않았다.
