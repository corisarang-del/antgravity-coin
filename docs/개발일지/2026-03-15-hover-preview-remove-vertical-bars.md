# hover 프리뷰 상하 여백 제거

- 작성시각: 2026-03-15 16:08 KST
- 해결하고자 한 문제:
  - 마우스를 올렸을 때 커지는 hover 프리뷰 안에서 상하 어두운 배경 여백이 크게 보였음
- 해결된 것:
  - hover 프리뷰 폭을 다시 넓힘
  - 미디어 프레임 비율을 `3:4`에서 원본에 더 가까운 `4:3`으로 조정함
  - 비디오와 이미지 폴백 내부 패딩을 제거해 상하 여백을 줄임
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 실제 mp4 원본 비율이 다르면 추가 미세 조정이 필요할 수 있음
