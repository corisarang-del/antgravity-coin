# 캐릭터도감 이미지 프레임 꽉 채움 보정

- 작성시각: 2026-03-15 15:02 KST
- 해결하고자 한 문제:
  - 캐릭터도감 카드에서 좌우 여백이 커 보여 이미지가 덜 차는 느낌이 남아 있었음
  - 상세 모달도 같은 톤으로 꽉 찬 느낌을 맞출 필요가 있었음
- 해결된 것:
  - 도감 카드 이미지 영역에 같은 캐릭터 이미지를 배경용 `cover + blur`로 깔아 프레임이 시각적으로 꽉 차 보이게 조정함
  - 실제 캐릭터 본체는 `contain`으로 유지해 비율을 지키면서 전부 보이게 유지함
  - 상세 모달 이미지 영역도 같은 방식으로 맞춤
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 캐릭터별 얼굴 위치나 시선 방향에 따라 더 세밀한 개별 오프셋 조정은 아직 하지 않음
