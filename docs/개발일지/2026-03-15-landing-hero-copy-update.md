# 랜딩 히어로 카피 수정

- 작성시각: 2026-03-15 16:50 KST
- 해결하고자 한 문제:
  - 랜딩 히어로 배지, 메인 카피, 설명 문장, 보조 버튼 라벨을 원하는 문구로 바꿔야 했음
  - 기존 파일 인코딩이 깨져 있어서 텍스트 수정이 불안정했음
- 해결된 것:
  - 랜딩 페이지 파일을 UTF-8 기준으로 정리함
  - 배지 문구를 `ANT GRAVITY`로 변경함
  - 메인 카피 하단을 `캐릭터 토론으로`로 변경함
  - 설명 문장을 삭제함
  - 보조 버튼 라벨을 `트레이더 먼저 보기`로 변경함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 없음
