# 랜딩 캐릭터 카드 정사각 미디어 프레임 적용

- 작성시각: 2026-03-15 16:22 KST
- 해결하고자 한 문제:
  - hover 프리뷰와 달리 랜딩 시즌 라인업 카드들은 아직 정사각 원본 비율에 맞춰지지 않았음
- 해결된 것:
  - 랜딩 시즌 라인업 카드의 미디어 프레임을 정사각형으로 조정함
  - 카드 폭을 소폭 넓혀 정사각 프레임이 답답하지 않게 보정함
  - 하단 이름/역할 소개영역은 그대로 유지함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 카드 높이와 라인업 전체 간격에 대한 추가 미세 튜닝은 아직 하지 않음
