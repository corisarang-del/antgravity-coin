# 캐릭터 이미지 비율 조정과 프리뷰 오버레이 제거

- 작성시각: 2026-03-15 14:35 KST
- 해결하고자 한 문제:
  - 캐릭터 도감 카드에서 이미지가 잘려서 캐릭터 전신이 다 보이지 않음
  - 캐릭터 설명 모달에서도 이미지가 잘려서 보임
  - 랜딩 캐릭터 라인업에서 hover 프리뷰 오버레이가 화면을 덮어 UI가 지저분해 보임
- 해결된 것:
  - `CharacterCard` 이미지를 `object-contain` 기준으로 바꿔 도감 카드에서 캐릭터가 최대한 전부 보이게 조정함
  - `CharacterDetailModal` 이미지를 `object-contain` 기준으로 바꿔 클릭 모달에서도 캐릭터가 잘리지 않게 조정함
  - 랜딩 `CharacterPosterCard`와 클릭 모달도 `object-contain` 기준으로 맞춰 전신 노출을 개선함
  - 랜딩 hover 프리뷰 오버레이를 제거해서 업로드 이미지에서 보이던 방해 요소를 없앰
  - 모달 테스트를 추가해 기본 정보 렌더링과 `object-contain` 클래스 적용을 검증함
- 해결되지 않은 것:
  - 캐릭터 원본 PNG 파일 자체 정리는 아직 하지 않음
  - 실제 크롭 만족도는 캐릭터별 원본 여백 차이에 따라 추가 미세조정이 필요할 수 있음
