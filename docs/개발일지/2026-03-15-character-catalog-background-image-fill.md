# 캐릭터도감 배경까지 캐릭터 이미지로 채우기

- 작성시각: 2026-03-15 15:10 KST
- 해결하고자 한 문제:
  - 캐릭터도감 카드와 모달에서 본체는 안 잘리지만 배경이 아직 비어 보였음
  - 배경까지 캐릭터 이미지로 채우되 이미지가 잘리지 않게 보이길 원했음
- 해결된 것:
  - 카드와 모달 이미지 영역에 같은 캐릭터 이미지를 좌우 복제 배경으로 배치함
  - 중앙 실제 캐릭터는 `contain`으로 유지해 비율을 지키면서 전부 보이게 유지함
  - 결과적으로 배경도 캐릭터 이미지로 채워지고 좌우 빈칸 체감이 줄어듦
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 캐릭터별 개별 구도 보정은 아직 공통 규칙만 적용됨
