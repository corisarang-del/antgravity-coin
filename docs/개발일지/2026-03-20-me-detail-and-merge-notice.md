# /me 상세 패널 연결과 병합 안내 UX 마감

작성시각: 2026-03-20 16:45 KST

## 해결하려고 한 문제

- `/me`에서 배틀 목록은 보여주지만 실제 상세 정보가 연결되지 않은 문제
- 로그인 직후 local 기록 병합이 사용자는 체감되지 않는 문제

## 해결된 것

- `/me?battleId=...` 기준으로 같은 페이지 안에서 상세 패널을 열 수 있게 만들었다.
- 상세 패널에서 정산 요약, 리포트, 토론 메시지, character memory를 함께 보여주도록 했다.
- 병합 성공/실패를 페이지 상단 안내 카드로 보여주게 했다.
- 전체 타입체크, 린트, 테스트를 다시 통과했다.

## 아직 안 된 것

- `/me` 상세 패널에서 리포트 소스 필터나 message 검색은 아직 없다.
- 병합 안내 카드는 현재 단발성 안내만 하고, 재시도 버튼은 없다.
