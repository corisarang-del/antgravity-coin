# 홈 검색 카드 외곽 overflow 수정

- 작성시각: 2026-03-23 23:03 KST
- 해결하고자 한 문제:
  - 업로드된 이미지 기준으로 홈 상단 오른쪽 검색 카드가 외곽 컨테이너 밖으로 밀려나 보였음
  - 원인은 한 줄 고정한 헤드라인과 grid 컬럼 최소 폭이 겹치면서 왼쪽 영역이 오른쪽 카드를 밀어내는 구조였음

## 해결된 것

- `src/app/home/page.tsx`
  - 상단 hero grid 컬럼을 `minmax(0, 1.5fr)` / `minmax(0, 1fr)` 형태로 바꿔 최소 폭이 0에서 줄어들 수 있게 조정했음
  - 왼쪽 컬럼에 `min-w-0`를 추가했음
  - `불리시팀 vs 베어리시팀` 헤드라인도 한 줄은 유지하되 크기를 조금 더 보수적으로 낮췄음
- `src/presentation/components/SearchBar.tsx`
  - 검색 카드 자체에 `min-w-0`를 추가했음
  - `어떤 코인으로 붙어볼지 골라줘` 헤드라인도 한 줄 유지 상태에서 크기를 한 단계 줄여 컨테이너 안쪽에 안정적으로 들어오게 조정했음
- 검증
  - `pnpm.cmd typecheck` 통과
  - 대상 파일 `eslint` 통과

## 해결되지 않은 것

- 실제 브라우저에서 이미지와 완전히 동일한 뷰포트로 시각 검수는 이번 턴에서 따로 하지 않았음
