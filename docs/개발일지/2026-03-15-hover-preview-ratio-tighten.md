# hover 프리뷰 비율 세로형 조정

- 작성시각: 2026-03-15 16:00 KST
- 해결하고자 한 문제:
  - hover 프리뷰 프레임이 아직 캐릭터 원본보다 가로로 넓어서 빈 공간 체감이 남아 있었음
- 해결된 것:
  - hover 프리뷰 최대 폭을 줄임
  - 미디어 프레임 비율을 `4:5`에서 `3:4`로 세로형에 더 가깝게 조정함
  - 내부 패딩을 줄여 캐릭터와 향후 mp4가 더 크게 보이게 조정함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 실제 mp4 소스별로 더 세밀한 비율 튜닝은 아직 하지 않음
