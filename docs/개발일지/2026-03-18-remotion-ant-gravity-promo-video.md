# Remotion 앤트그래비티 소개 영상 추가

- 작성시각: 2026-03-18 22:55 KST
- 해결하고자 한 문제: 현재 프로젝트 구조를 바탕으로 앤트그래비티의 핵심 가치와 후킹 포인트를 짧고 강렬하게 소개하는 코드 기반 영상을 만들고 싶었다. 기존 저장소에는 Remotion 구성이 없어 영상 제작과 렌더링을 바로 시작할 수 없는 상태였다.

## 이번 단계에서 해결한 것

- `remotion`, `@remotion/cli`, `@remotion/media`를 프로젝트에 추가했다.
- `src/remotion` 아래에 엔트리, Root Composition, 소개 영상 컴포넌트를 새로 만들었다.
- 앤트그래비티 소개 카피와 씬 구조를 `content.ts`로 분리하고 테스트를 추가했다.
- 소개 영상은 7초, 1920x1080, 30fps 기준으로 구성했다.
- 랜딩과 캐릭터 구조를 반영해서 8명의 캐릭터가 한 번에 보이는 라인업 장면을 넣었다.
- `pnpm video:studio`, `pnpm video:render` 스크립트를 추가했다.

## 아직 해결되지 않은 것

- 실제 MP4 렌더링 결과의 시각 퀄리티는 프리뷰와 렌더 결과를 보고 한 번 더 다듬을 여지가 있다.
- 배경 음악, 효과음, 나레이션은 아직 넣지 않았다.
- Remotion MCP는 현재 세션에서 사용 불가능해서 공식 문서 실시간 검색 기반 보강은 하지 못했다.
