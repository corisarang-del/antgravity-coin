# 내 페이지 등급 타이틀 canonical fix

- 작성시각: 2026-03-30 23:20 KST

## 해결하고자 한 문제

- `/me` 상단 헤더 쪽은 `개미`가 정상인데, `현재 등급` 카드만 깨진 문자열로 보이는 케이스가 남아 있었음
- 이건 저장된 `title` 문자열을 그대로 신뢰하는 경로가 남아 있을 가능성보다, title 자체를 DB/localStorage 값에 의존하는 설계가 불필요하게 취약한 게 더 큰 원인으로 판단했음

## 해결된 것

- `normalizeUserLevel()`가 저장된 `title` 문자열을 더 이상 신뢰하지 않고 `level` 기준 canonical title만 사용하도록 변경했음
- 이제 `level=1..5`만 맞으면 항상 `개미`, `새싹개미`, `중급개미`, `고수개미`, `전설개미` 중 하나로 고정됨
- 깨진 문자열, 오래된 문자열, 임의 문자열이 저장돼 있어도 `/me`에서는 항상 정상 등급명으로 보정됨
- `levelMapper.test.ts`에 임의 문자열도 canonical title로 교정되는 테스트를 추가했음

## 아직 안 된 것

- 실제 로그인 세션에서 preview 화면 재확인은 사용자가 한 번 더 확인해줘야 함
