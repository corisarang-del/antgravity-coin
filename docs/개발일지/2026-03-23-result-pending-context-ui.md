# 결과 페이지 pending 문맥 UI 추가

- 작성시각: 2026-03-23 20:19 KST
- 해결하고자 한 문제:
  - `waiting`에서 `결과 화면 열기`를 누르면 결과 페이지로 이동하긴 하지만, 정산 전에는 카운트다운만 보여서 버튼이 그냥 사라진 것처럼 느껴졌음
  - 실제로는 결과 페이지에 들어왔다는 문맥과 지금 무엇을 기다리는지 설명이 부족했음

## 해결된 것

- `src/app/battle/[coinId]/result/ResultPageClient.tsx`
  - `settlementPending` 상태에서도 `MyPickSummary`를 같이 보여주도록 추가했음
  - `결과 페이지 준비 중` 설명 섹션을 추가했음
  - 아래 3단계 진행 문맥을 같이 보여주도록 했음
    - 차트 마감 대기
    - 승패와 XP 계산
    - 리포트와 요약 정리
  - 현재까지 저장된 발언 수(`n/8`)도 함께 보여주도록 추가했음
- 검증
  - `pnpm run typecheck` 통과
  - 대상 파일 eslint 통과

## 해결되지 않은 것

- 버튼 라벨 자체는 여전히 `결과 화면 열기`라서 완전한 UX 정합성까지 맞춘 건 아님
- waiting/result 전환을 브라우저로 다시 끝까지 실측한 건 아니고, 이번엔 결과 pending UI 맥락 보강에 집중했음
