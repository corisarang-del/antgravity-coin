# 캐릭터 모달 단일 원본 이미지 꽉 채우기

- 작성시각: 2026-03-15 16:32 KST
- 해결하고자 한 문제:
  - 캐릭터도감 모달 상단에서 배경에 이미지를 복제해 채우는 방식은 가독성을 떨어뜨렸음
  - 배경 없이 원본 이미지 하나만으로 꽉 차게 보여주고 싶었음
- 해결된 것:
  - 모달 상단 미디어 프레임을 정사각형으로 맞춤
  - 배경 이미지 복제 레이어를 제거함
  - 원본 이미지를 `object-cover`로 단독 표시해 프레임을 꽉 채우도록 변경함
  - `pnpm lint`, `pnpm typecheck`, `pnpm test -- CharacterDetailModal` 통과
- 해결되지 않은 것:
  - 없음
