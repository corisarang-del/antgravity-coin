# 개발서버 포트 정리 명령 추가

- 작성시각: 2026-03-15 10:40 KST
- 해결하고자 한 문제: `next dev` 재실행 시 `.next/dev/lock`과 기존 포트 점유 프로세스 때문에 개발 서버가 시작되지 않는 문제를 빠르게 정리할 명령이 필요했음.
- 해결된 것:
  - `3001` 포트를 점유한 개발 서버 프로세스를 종료함.
  - `scripts/kill-dev-port.ps1`를 추가해 포트 점유 프로세스 종료와 락 파일 삭제를 한 번에 처리하도록 함.
  - `package.json`에 `pnpm run dev:clean` 명령을 추가함.
- 해결되지 않은 것:
  - 사용자 환경에서 `pnpm` 대신 `pnpm.cmd`를 직접 써야 하는 PATH 문제는 별도로 남아 있음.
