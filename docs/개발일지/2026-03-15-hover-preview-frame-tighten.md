# hover 프리뷰 프레임 축소 정리

- 작성시각: 2026-03-15 15:53 KST
- 해결하고자 한 문제:
  - hover 프리뷰에서 남는 영역을 숨기는 대신 프리뷰 프레임 자체를 더 타이트하게 줄여 이미지가 꽉 차 보이게 해야 했음
- 해결된 것:
  - hover 프리뷰의 배경 캐릭터 이미지 레이어를 제거한 상태에서 프레임 안쪽을 더 단순한 어두운 그라데이션으로 정리함
  - 실제 비디오와 이미지 폴백은 `contain`을 유지해 잘리지 않게 둠
  - 남는 영역은 캐릭터 배경 복제 없이 프레임 자체 안에서만 처리되게 정리함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 실제 mp4 에셋을 넣은 상태에서 개별 캐릭터별 세밀한 프레임 튜닝은 아직 하지 않음
