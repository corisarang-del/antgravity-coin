# 로그인 헤드라인 한 줄 튜닝

- 작성시각: 2026-03-23 22:18 KST
- 해결하고자 한 문제:
  - 로그인 페이지 헤드라인 `로그인 / 한번이면 / 끝`도 여전히 설명성이 남아 있고 줄 수가 많아 보였음
  - 더 짧고 한 줄인 문장으로 정리하면서 기존 심플 톤과 타이포 균형을 유지해야 했음

## 해결된 것

- `src/app/login/LoginPageClient.tsx`의 메인 헤드라인을 `바로 시작해` 한 줄로 변경했음
- 한 줄 헤드라인이 답답해 보이지 않게 크기, 자간, 최대 너비를 같이 조정했음
- 검증
  - `node_modules\.bin\eslint.cmd src/app/login/LoginPageClient.tsx` 통과

## 해결되지 않은 것

- 실제 브라우저에서 최종 시각 밸런스 확인은 이번 턴에서 따로 하지 않았음
- 나머지 로그인 페이지 카피를 더 줄일지는 다음 피드백에 따라 추가 조정 가능함
