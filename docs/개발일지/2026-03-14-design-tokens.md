# 개발일지 - 프로젝트 전용 디자인 토큰 생성

작성시각: 2026-03-14 14:33 KST

## 해결하려던 문제

- 현재 프로젝트에 맞는 디자인 토큰을 만들고
- `shadcn/ui`에서 바로 사용할 수 있는 CSS 변수 구조로 정리하려고 했다

## 진행 내용

- 기존 레퍼런스 분석 내용을 바탕으로 제품 성격에 맞는 컬러 체계를 재설계했다
- `src/styles/tokens.css`에 라이트/다크 테마 토큰을 작성했다
- `bull`, `bear`, `xp` 같은 도메인 전용 상태색을 분리했다
- 토큰 설계와 검토 결과를 문서로 남겼다

## 해결된 것

- `shadcn/ui` 호환 시맨틱 토큰 세트를 만들었다
- 모바일 우선 간격, 라운드, 타이포 스케일 기준을 정했다
- 다크 모드 토큰까지 포함했다

## 아직 안 된 것

- 실제 컴포넌트에 토큰 적용
- 프리뷰 페이지 제작
- 버튼/카드/배지 실물 대비 검증

## 메모

- 현재 저장소에 앱 코드가 없어 토큰 파일을 먼저 공용 자산으로 만들었다
