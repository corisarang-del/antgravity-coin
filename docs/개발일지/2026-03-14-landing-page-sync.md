# 개발일지 - 랜딩 페이지 추가와 디자인 동기화

작성시각: 2026-03-14 20:45 KST

## 해결하려던 문제

- 첫 홈 화면 전에 강한 첫 인상을 주는 랜딩 페이지를 만들고
- 그 구조를 넷플릭스형으로 가져가되 색감과 공통 컴포넌트 톤은 참고 이미지에 맞추려고 했다

## 진행 내용

- `/`를 랜딩 페이지로 교체했다
- 기존 홈 화면은 `/home`으로 이동했다
- 랜딩 전용 토큰을 `src/styles/tokens.css`에 추가했다
- `design/MASTER.md`를 넷플릭스형 랜딩 + 안트그래비티 톤 기준으로 다시 정리했다

## 해결된 것

- 강한 첫 인상용 랜딩 구조가 생겼다
- 홈 진입 전 브랜드 경험을 분리했다
- 디자인 문서와 코드 토큰이 같은 방향으로 맞춰졌다

## 아직 안 된 것

- 넷플릭스 참고 이미지 기반 세부 애니메이션 조정
- 랜딩 전용 카드 hover 모션 추가
- refer 폴더 기반 더 세밀한 카피 큐레이션
