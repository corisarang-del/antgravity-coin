# 캐릭터 도감 초보용 해설 추가

- 작성시각: 2026-03-28 21:08 KST

## 해결하고자 한 문제

- 캐릭터 도감의 전문 용어가 익숙한 사람 기준으로만 적혀 있어서 초보가 봤을 때 RSI, 공포탐욕, 온체인 같은 표현을 바로 이해하기 어려웠음

## 해결된 것

- 캐릭터 데이터 모델에 `beginnerSummary`, `beginnerGuide`를 추가했음
- 8명 전 캐릭터에 대해 쉬운 한줄 요약과 용어별 `쉬운 뜻`, `왜 중요해?` 해설을 채웠음
- 캐릭터 카드에는 `초보용 한줄 해설`을 추가해서 도감 목록에서 바로 이해되게 했음
- 사용자 피드백 반영으로 상세 모달에서는 `고른 이유`를 완전히 제거하고 `성격`을 다시 중심 정보로 복구했음
- 상세 모달의 초보 해설은 `어려운 부분만 쉽게 보기` 섹션으로 바꾸고, 캐릭터마다 1~2개 어려운 포인트만 남기도록 줄였음
- `src/app/characters/CharactersPageClient.tsx` 소개 문구와 planning 문서도 현재 UI 기준으로 같이 맞췄음
- `/api/characters` 응답 스키마도 같은 필드를 포함하도록 맞췄음
- API 테스트와 도감 UI 테스트를 같이 보강했음
- `pnpm.cmd typecheck` 통과
- 대상 파일 `eslint` 통과

## 아직 안 된 것

- Vitest는 권한 올려 실행해야 하는 환경이라 기본 sandbox에선 `spawn EPERM`이 날 수 있음
- 이번 변경은 `캐릭터 도감(/characters)`에만 적용했고, 랜딩 캐릭터 레일이나 배틀 화면 문구는 건드리지 않았음
