# admin 메모 아카이브 분리와 lesson prompt 고도화

- 작성시각: 2026-03-18 22:41 KST
- 해결하고자 한 문제:
  - admin에서 재사용 메모를 배틀 상세와 분리해서 따로 보고 싶었다.
  - Gemini lesson JSON 품질을 더 올리기 위해 캐릭터별 길이 제한과 금지어까지 추가하고 싶었다.
  - 구현 후 실제 브라우저에서 목록-상세 흐름과 모바일 레이아웃을 확인해야 했다.

## 해결된 것
- `/admin/memos` 전용 화면을 추가해 재사용 메모 목록과 상세를 따로 볼 수 있게 했다.
- admin 상단에 `배틀 대시보드` / `재사용 메모` 섹션 네비게이션을 추가했다.
- lesson JSON 프롬프트에 캐릭터별 focus/금지어, 길이 제한, 출력 개수 제한을 넣어 더 타이트하게 조였다.
- 브라우저에서 `/admin/memos` empty state, 샘플 데이터 생성 후 목록/상세, 모바일 가로 스크롤 여부까지 확인했다.
- `pnpm lint`, `pnpm typecheck`, 관련 `vitest`를 통과했다.

## 아직 해결 안 된 것
- `database/data/*.json`에 들어간 샘플/운영 데이터 자체는 이번 커밋에 포함하지 않았다.
- admin 메모 화면에 정렬/필터/검색 확장까지는 아직 하지 않았다.
