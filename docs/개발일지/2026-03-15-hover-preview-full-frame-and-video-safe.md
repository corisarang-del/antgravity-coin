# hover 프리뷰 전신 노출과 비디오 안전 프레임 조정

- 작성시각: 2026-03-15 15:42 KST
- 해결하고자 한 문제:
  - hover 프리뷰도 캐릭터가 더 크게 보여야 했고, mp4를 나중에 넣었을 때 잘리지 않게 해야 했음
  - 프리뷰 배경은 비어 보이지 않게 캐릭터 이미지 기반으로 꽉 차 보이길 원했음
- 해결된 것:
  - hover 프리뷰 박스 폭을 키우고 세로형 비율로 조정함
  - 영상은 `object-contain`으로 변경해서 mp4 추가 시 전신이 안 잘리게 함
  - 배경에는 캐릭터 이미지를 깔아 프리뷰 프레임이 비어 보이지 않게 맞춤
  - 영상이 없을 때도 같은 프레임에서 이미지 프리뷰가 자연스럽게 보이게 유지함
  - `pnpm lint`, `pnpm typecheck` 통과
- 해결되지 않은 것:
  - 실제 캐릭터별 mp4 에셋은 아직 추가되지 않음
