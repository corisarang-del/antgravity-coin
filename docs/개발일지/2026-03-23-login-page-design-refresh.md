# login page design refresh

- 작성시각: 2026-03-23 21:48 KST
- 해결하고자 한 문제:
  - 로그인 페이지가 현재 디자인 규칙에 비해 카드 계층과 강조 포인트가 약해서 첫 인상이 다소 평평했음
  - 헤더의 `로그인/회원가입` 버튼이 다른 상단 버튼들과 시각적 톤이 완전히 맞지는 않았음

## 해결된 것

- `src/app/login/LoginPageClient.tsx`

  - 로그인 후 얻게 되는 정보 구조를 보조 카드로 더 명확히 보여주게 했음
- `src/presentation/components/AppHeader.tsx`
  - `로그인/회원가입` CTA를 다른 헤더 버튼과 더 조화되는 카드형 스타일로 조정했음
- 이번 변경 기록을 프롬프트 문서와 개발일지에 함께 남겼음
